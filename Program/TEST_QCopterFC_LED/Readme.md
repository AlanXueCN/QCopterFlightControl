QCopterFC LED Example
========
* Author  : [Hom](https://github.com/Hom19910422)
* Update  : 2014/01/17

Description
========
QCopterFC LED 和 KEY 的範例程式，啟動時 LED 會依序閃爍 ( R → G → B )，當按下按鍵 KEY 時，LED 則會同時閃爍，放開按鍵又會依序閃爍。

Hardware
========
* LED  
PC13 - LED_B  
PC14 - LED_G  
PC15 - LED_R  

* KEY  
PB2  - KEY
