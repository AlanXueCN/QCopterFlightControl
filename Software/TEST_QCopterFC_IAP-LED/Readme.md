QCopterFC IAP-LED Example
========
* Author  : [Hom](http://about.me/Hom)
* Update  : 2014/07/08

Description
========
QCopterFC IAP 的 LED 和 KEY 範例程式，啟動時 LED 會依序閃爍 ( R → G → B )，當按下按鍵 KEY 時，LED 則會同時閃爍，放開按鍵又會依序閃爍。  
 > 詳細參考 TEST_QCopterFC_LED

Hardware
========
* LED  
PC13 - LED_B  
PC14 - LED_G  
PC15 - LED_R  

* KEY  
PB2  - KEY
