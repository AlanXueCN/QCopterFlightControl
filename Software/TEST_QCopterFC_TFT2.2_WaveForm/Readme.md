QCopterFC TFT2.2_WaveForm Example
========
* Author  : [Hom](about.me/Hom)
* Update  : 2014/06/05

Description
========
QCopterFC TFT2.2_WaveForm 的範例程式，透過 FFCSPI 讀寫 2.2 吋螢幕，計算三角函數並將波形顯示出來。

Hardware
========
* LED  
PC13 - LED_B  
PC14 - LED_G  
PC15 - LED_R  

* KEY  
PB2  - KEY  

* FFCSPI  
PA4 - CST  
PA5 - SCK  
PA6 - SDO  
PA7 - SDI  
PC0 - RST  
PC1 - DC
