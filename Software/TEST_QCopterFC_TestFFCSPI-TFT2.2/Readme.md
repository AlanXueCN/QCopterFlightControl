QCopterFC FFCSPI-TFT2.2 Example
========
* Author  : [Hom](about.me/Hom)
* Update  : 2014/05/06

Description
========
QCopterFC FFCSPI-TFT2.2 的範例程式，透過 FFCSPI 讀寫 2.2 吋螢幕，並顯示彩條。

Hardware
========
* LED  
PC13 - LED_B  
PC14 - LED_G  
PC15 - LED_R  

* KEY  
PB2  - KEY  

* FFCSPI  
PA4 - CST  
PA5 - SCK  
PA6 - SDO  
PA7 - SDI  
PC0 - RST  
PC1 - DC
