QCopterFC PWM Example
========
* Author  : [Hom](about.me/Hom)
* Update  : 2014/05/06

Description
========
QCopterFC PWM 的範例程式，按下按鍵 KEY 時，會改變 PWM 的 Duty Cycle，LED 同時會閃爍。

Hardware
========
* LED  
PC13 - LED_B  
PC14 - LED_G  
PC15 - LED_R  

* KEY  
PB2  - KEY  

* PWM  
PA0  - TIM2 CH1 - PWM1  
PA1  - TIM2 CH2 - PWM2  
PA2  - TIM2 CH3 - PWM3  
PA3  - TIM2 CH4 - PWM6  
PB0  - TIM3 CH3 - PWM5 
PB1  - TIM3 CH4 - PWM4  
PB8  - TIM4 CH3 - PWM10  
PB9  - TIM4 CH4 - PWM11  
PC6  - TIM3 CH1 - PWM8  
PC7  - TIM3 CH2 - PWM7
